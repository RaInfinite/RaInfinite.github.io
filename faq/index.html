FAQ

  • How can I get an update when a new post comes out?

    Use the RSS feed for now. A newsletter can wait until the blog has a real writing rhythm.

  • What if I see something incorrect in a post?

    Send a note by email. Corrections are welcome, especially when they make the reasoning clearer.

  • Can posts include images?

    Yes. Images live under assets/images/, so the site can keep them inside the GitHub Pages repo.

  • Do old posts get updated?

    Yes. If a post changes meaningfully, add a short update note near the top of that post.

  • How can you keep up blogging?

    By treating it as a thinking habit, not a publishing business.
